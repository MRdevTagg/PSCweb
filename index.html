Identidad Visual:
Definir un Identidad Visual coherente con lo que la empresa vende y respetarlo a la hora de crear contenido.

Optimizacion:
Ajustar y optimizar el material gráfico para utilizarlo en las composciciones visuales realizadas para el sitio y/o otras redes. Támbien en esta etapa se trabajaran aspectos como animaciónes y el diseño de la interfaz de usuario (menús, navegacion, etc)

Diseño Responsive :
Esto significa que se aplicará un diseño escalable que se adapte a varias resoluciones de pantalla para visualizarlo de forma correcta en multiples dispositivos ( celulares, PC, Tablets, notebooks, etc )

Objetivos Globales:
Definir La estructura del Sitio Web buscando la mejor manera de mostrar el contenido, que sea accesible, comodo para los usuarios y los lleve a seguir navegando e informandose sobre los servicios que se ofrecen.....

....Es importante en este punto que los usuarios puedan contactarse en cualquier momento mientras esten en el sitio, Una buena solución es dejar fijado un boton en pantalla que los lleve a un contacto directo

Evaluar como Gestionar el contenido ( si sera autogestionado y si se nececita capacitación)

Obtener un Dominio y un Hosting + Certificado SSL : (Por lo general viene todo junto, las tres cosas son fundamentales para subir un Sitio Web de forma Segura, Si es de interes se puede entrar en detalle)

Plazos : Evaluar y fijar un plazo conveniente
